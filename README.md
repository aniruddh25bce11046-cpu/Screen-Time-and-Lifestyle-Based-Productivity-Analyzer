# Screen-Time-and-Lifestyle-Based-Productivity-Analyzer

# Overview
This project is a rule-based intelligent system that analyzes daily activities such as screen time, sleep, study, exercise, and meal duration to predict a user's productivity level.

# Problem Statement
In today's digital world, excessive screen time and poor lifestyle habits negatively affect productivity. This project aims to evaluate how daily habits influence productivity and provide useful suggestions.

##  Solution
The system takes user input for daily activities such as sleep, study,  screentime, exercise, mealtime and applies rule-based logic to calculate a productivity score. Based on this score, the system classifies productivity as:
- High
- Medium
- Low

It also provides suggestions to improve daily routine.

# Features
- Input validation (ensures total time ≤ 24 hours)
- Productivity prediction using scoring logic
- Personalized suggestions to improve productivity
- It also calculate free time left by the user

# Technologies Used
- Python (No external libraries)
- Rule-Based AI Logic

# How to Run
1. Install Python
2. Download the project file
3. Run the program:

ScreenTimeProductivity_analyzer.py

4. Enter your daily activity hours 

##  Example Output
- Productivity Level: High
- Score: 6
- Suggestions: Improve sleep, reduce screen time, etc.
